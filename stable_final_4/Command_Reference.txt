================================================================================ RS-232 COMMAND SUMMARY (Direct Commands)
Command | Function

K0        | Disable battery backup
K1        | Select valve to OPEN upon power fail
K2        | Select valve to CLOSE upon power fail

NOTES

Note 1 (R7 - MXYZ):
X: 0=Analog, 1=A, 2=B, 3=C, 4=D, 5=E
Y: 0=Controlling, 2=Valve Open, 4=Valve Close
Z: 0=Pressure < 10% F.S., 1=Pressure > 10% F.S.

Note 2 (R23 - J type):
1 = Std 253, 2 = Fast 253, 3 = 653

Note 3 (R33 - E value):
00=0.1, 01=0.2, 02=0.5, 03=1, 04=2, 05=5, 06=10, 07=50, 08=100,
09=500, 10=1000, 11=5000, 12=10000

Note 4 (R34 - F value):
00=Torr, 01=mTorr, 02=mbar, 03=Âµbar, 04=kPa, 05=Pa, 06=cmH2O, 07=inH2O

Note 5 (R37 - MXYZ):
X: 0=Local, 1=Remote
Y: 0=Not Learning, 1=Learning, 2=Learning Valve System
Z: 0=Open, 1=Close, 2=Stop, 3=SP A, 4=SP B, 5=SP C, 6=SP D, 7=SP E, 8=Analog SP
